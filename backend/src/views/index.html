API RESTful - DnD5e (MongoDB)

API con autenticacion JWT y CRUD para Spells, Monsters, Races y Equipment importados desde D&D 5e API.

Endpoints

Spells

  • GET /api/spells (filtros: ?name=, ?level=, ?school=, ?ritual=, ?concentration=, paginacion ?page=&limit=)
  • GET /api/spells/:id
  • POST /api/spells
  • PATCH /api/spells/:id
  • DELETE /api/spells/:id

Monsters

  • GET /api/monsters (filtros: ?name=, ?type=, ?crMin=, ?crMax=, paginacion ?page=&limit=)
  • GET /api/monsters/:id
  • POST /api/monsters
  • PATCH /api/monsters/:id
  • DELETE /api/monsters/:id

Races

  • GET /api/races (filtros: ?name=, ?size=, paginacion ?page=&limit=)
  • GET /api/races/:id
  • POST /api/races
  • PATCH /api/races/:id
  • DELETE /api/races/:id

Equipment

  • GET /api/equipment (filtros: ?name=, ?category=, ?gear=, ?minWeight=, ?maxWeight=, paginacion ?page=&limit=)
  • GET /api/equipment/:id
  • POST /api/equipment
  • PATCH /api/equipment/:id
  • DELETE /api/equipment/:id

Como empezar

npm i
cp .env.example .env
# Editar MONGO_URI si hace falta
npm run seed
npm run dev

API RESTful – DnD5e (MongoDB)

Esta API expone recursos de Spells y Monsters importados desde D&D 5e API.

Endpoints

Spells

  • GET /api/spells (filtros: ?name=, ?level=, ?school=, ?ritual=, ?concentration=, paginación ?page=&limit=)
  • GET /api/spells/:id
  • POST /api/spells
  • PATCH /api/spells/:id
  • DELETE /api/spells/:id

Monsters

  • GET /api/monsters (filtros: ?name=, ?type=, ?crMin=, ?crMax=, paginación ?page=&limit=)
  • GET /api/monsters/:id
  • POST /api/monsters
  • PATCH /api/monsters/:id
  • DELETE /api/monsters/:id

Cómo empezar

npm i
cp .env.example .env
# Editar MONGO_URI si hace falta
npm run seed
npm run dev
